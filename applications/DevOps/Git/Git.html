Git

Le concept

Git est un logiciel de gestion de versions décentralisé. C'est un logiciel libre créé par Linus Torvalds, auteur du noyau Linux
Les commandes à connaitre

Commande

Objectif

git init Pour activer un dossier comme repository Git
git add mon_fichier Pour ajouter à l'index mon_fichier
git add . Pour ajouter tous les fichiers dans le répertoire courant
git status Pour afficher l'état des commits
git commit -m "initialisation site" Pour enregistrer vos modifications
git commit --amend -m "Votre nouveau message" Pour modifier le message de votre dernier commit
git log Pour afficher les commits avec leur SHA : les identifiants uniques des commits
git clone https://github.com/facebook/react.git Pour télécharger un projet
git pull origin master Pour ne récupérer que les modifications d'un tiers
git remote add origin https://github.com/jeg56/monsite.git Pour ajouter une nouvelle machine
git push --force origin master Pour envoyer un commit sur githup (origine etant la machine et le master la branche)
git remote -v Pour afficher les dépôts distants
git branch Pour afficher toutes les branches disponibles dans mon repository
git branch nouvelle-branche Pour créer une nouvelle branche
git checkout nouvelle-branche Pour se placer dans une autre branche à l'intérieur de votre repository
git checkout SHADuCommit Pour se positionner sur un commit
git revert SHADuCommit Pour créer un commit qui fait l'inverse du précédent
git reset --hard‌ Pour annuler tous les changements non commités
git merge brancheB Pour mettre à jours la branche positionné par rapport à brancheB
git blame nomFichier Pour connaitre l'origine de la modification de chaque ligne (user + SHADuCommit)
Git show SHADuCommit Pour afficher les modifications
git stash Pour mettre ses modifications de coté
git stash pop Pour récupérer ses modifications
fork Pour faire une copie d'un projet distant et le mettre sur son compte
